GnuDIP Release 2.3 - Windows Client INSTALL File

Follow these steps:

  1. Move the folder gdipc created when you expanded Win_gdipc.exe, and containing this INSTALL.html file, to C:\.

  2. You may wish to add the folder C:\gdipc to your executable search path, either:

    • by adding a line to your C:\autoexec.bat file such as this: 
      path "%PATH%";"C:\gdipc"
      You will need to restart Windows in order to have this take effect.

    • or by using the GUI interface that Windows NT/2000 provides for this purpose:
      1. Open your NT system's environment settings:

        1. Double-click the My Computer icon on your desktop.
        2. Double-click the Control Panel icon in the My Computer folder.
        3. Double-click the System icon in the Control Panel folder.
        4. Click the Environment tab in the System Properites window.

      2. In the System Variables window, find the variable Path and click it. Notice that the variable and its value appear in the two text fields at the bottom of the display.

  3. Read CLIENT.html for a description of how to use this GnuDIP Windows client now that it is installed.
