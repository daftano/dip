GnuDIP Release 2.3 - Linux/UNIX Client INSTALL File

Follow these steps:

  1. Move the directory gdipc created when you expanded the client tar ball, and containing this INSTALL.html file, to /usr/local.

  2. You may wish to add the folder /usr/local/gdipc/bin to your executable search path.

  3. Read CLIENT.html for a description of how to use this now that it is installed.
